# CLAUDE.md

## Goal

Create a small React app that acts as a simple browser synthesizer.

## Requirements

* oscillator waveforms: sine, square, triangle, sawtooth
* a filter control
* a small playable keyboard UI

## Tech

* React
* Web Audio API
* minimal dependencies

## Principles

* keep the project small and readable
* prefer simple solutions over abstractions
* separate audio logic from UI
* use modern React patterns

## Out of Scope

Do not add advanced synth features (MIDI, presets, sequencers, effects chains).
